# YWT Attendence
A Simple Electron-Based Attendence System

## Installation
 - Clone the Repository ```git clone http://github.com/Sacrafex/YWT-Attendence-System.git```
 - Modify ```index.html``` on lines ```214 - 217``` for admin actions
 - run ```make``` and open finished package in ```/dist```

## Features
  - Barcode Scanning System with Extra Hours and Session Hours
  - Admin Panel with User Actions
  - Intuitive Design & UI
  - Simplicity
